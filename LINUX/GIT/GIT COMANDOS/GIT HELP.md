# GIT HELP
O comando `git help` é usado para acessar a documentação dos subcomandos do git, como `git commit`, `git log`, `git branch`, entre outros. Ele permite entender **como usar um comando**, **quais opções ele tem** e **exemplos de uso no terminal.**

![[git help.png]]


### Ver informações de um comando específico
```bash
git help [comando]
```

Exemplo:
```bash
git help log
```
Esse comando abre a documentação do `git log` explicando como ele funciona, suas opções e exemplos práticos.

### Configuração visualizador
- Configurar visualizador para o navegador
```bash
git config --global help.format web #Abre a documentação no navegador
```
