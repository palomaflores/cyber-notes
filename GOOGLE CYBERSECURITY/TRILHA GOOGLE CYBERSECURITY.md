# Google Cybersecurity
Este arquivo contém a **trilha completa do programa de certificação em cibersegurança**, com uma visão geral de cada etapa do aprendizado e os **objetivos específicos de cada curso**. É um guia organizado para acompanhar a evolução dos estudos e entender o foco de cada módulo dentro do programa.

#### 1. Fundamentos de Cibersegurança
- Reconhecer as principais habilidades e conhecimentos necessários para se tornar um analista de segurança cibernética
- Identificar como os ataques à segurança afetam as operações comerciais
- Explicar a ética de segurança
- Identificar ferramentas comuns usadas por analistas de segurança cibernética

### 2. Gerenciamento de Riscos
- Identificar as principais ameaças, riscos e vulnerabilidades às operações comerciais
- Examinar como as organizações usam estruturas e controles de segurança para proteger as operações
- Definir ferramentas de gerenciamento de eventos e informações de segurança (SIEM)
- Usar um manual para responder a ameaças, riscos e vulnerabilidades

### 3. Redes de Computadores e Segurança de Rede
- Definir os tipos de redes e os componentes das redes
- Ilustrar como os dados são enviados e recebidos em uma rede
- Compreender como proteger uma rede contra táticas de invasão
- Descrever as técnicas de proteção do sistema

### 4. Linux e SQL
- Explicar a relação entre sistemas operacionais, aplicativos e hardware
- Comparar uma interface gráfica de usuário com uma interface de linha de comando
- Navegar e gerenciar o sistema de arquivos usando comandos do Linux por meio do shell Bash
- Usar o SQL para recuperar informações de um banco de dados

### 5. Recursos, Ameaças e Vulnerabilidades
- Classificar os ativos
- Analisar uma superfície de ataque para encontrar riscos e vulnerabilidades
- Identificar ameaças como engenharia social, malware e explorações baseadas na web
- Resumir o processo de modelagem de ameaças

### 6. Detecção e Resposta
- Identificar as etapas para conter, erradicar e se recuperar de um incidente
- Analisar pacotes para interpretar comunicações de rede
- Compreender a sintaxe básica e os componentes de assinaturas e registros em IDS e NIDS
- Realizar consultas em ferramentas SIEM para investigar eventos

### 7. Automação de Tarefas com Python
- Explicar como a linguagem Python é usada na segurança cibernética
- Criar funções Python definidas pelo usuário
- Usar expressões regulares para extrair informações de texto
- Praticar a depuração de código

### 8. Do Curso para o Trabalho
- Determinar quando e como escalar um incidente de segurança
- Envolver-se com a comunidade de segurança cibernética
- Candidatar-se a empregos e se preparar para entrevistas
- Obter vantagem competitiva com habilidades de IA ensinadas por especialistas do Google
